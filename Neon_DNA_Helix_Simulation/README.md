# Neon DNA Helix Simulation

Neon DNA Helix Simulation is a fullscreen visual animation created with HTML, CSS and JavaScript.

It creates a futuristic rotating DNA-style helix with neon nodes, glowing connection lines, star particles and mouse-reactive motion.

## Project Purpose

The purpose of this project is to create a code-based animated visual simulation suitable for GitHub, LinkedIn, YouTube Shorts and Instagram/Reels content.

This is not a dashboard or a standard web application.  
It is a pure visual simulation focused on 3D motion, neon effects and creative coding.

## Main Features

- Fullscreen DNA helix animation
- Neon glowing nodes
- Double helix connection lines
- 3D depth projection
- Mouse-reactive rotation
- Star particle background
- Glowing energy core
- Smooth animation loop
- Pure HTML, CSS and JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API
- requestAnimationFrame
- Custom 3D projection math

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.
4. The visual simulation starts automatically.
