# Neon DNA Helix Simulation

Neon DNA Helix Simulation is a fullscreen visual animation project created with HTML, CSS, and JavaScript.

This project is not a dashboard or a standard web application. It is a pure visual simulation focused on 3D motion, neon effects, double helix structure, depth projection, and interactive creative coding.

---

## Project Purpose

The purpose of this project is to create a futuristic DNA helix animation using only frontend technologies.

The simulation creates a rotating DNA-style double helix with glowing neon nodes, connection lines, star particles, depth motion, and mouse-reactive rotation. It is designed for GitHub portfolio, LinkedIn sharing, YouTube Shorts, and Instagram/Reels content.

---

## Main Features

- Fullscreen visual simulation
- Neon DNA helix animation
- Double helix structure
- Glowing node particles
- Neon connection lines
- 3D depth projection
- Mouse-reactive rotation
- Star particle background
- Glowing energy core
- Smooth animation loop
- Dark futuristic background
- Pure HTML, CSS, and JavaScript

## Technologies Used
